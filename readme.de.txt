RSS Feed Reader for Homey

Diese App ermöglicht es Homey, auf den von Ihnen eingerichteten RSS-Feed zu reagieren.
- Verwenden Sie es in Flows, um Aktionen auszulösen, zum Beispiel, dass Homey Ihnen aktuelle Ereignisse mitteilt.

Verwendung ⚙
- Installieren Sie diese App auf Ihrem Homey.
- Fügen Sie den RSS-Feed Reader als neues Gerät hinzu.
- Gehen Sie zu den Geräteeinstellungen und geben Sie einen RSS-Feed ein.

Aktuelle Funktionen 🔧
- [x] Echtzeit-Benachrichtigungen (vergessen Sie nicht, einen Stream zu erstellen)
