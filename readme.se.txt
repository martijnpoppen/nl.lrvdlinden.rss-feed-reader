RSS Feed Reader For Homey

Denna app gör det möjligt för Homey att reagera på det RSS-flöde du har konfigurerat.
- Använd den i flöden för att utlösa åtgärder, till exempel, få Homey att tala för att informera dig om vad som händer.

Så här använder du ⚙
- Installera denna app på din Homey.
- Lägg till RSS-flödesläsare som en ny enhet.
- Gå till enhetens inställningar och ange ett RSS-flöde.

Nuvarande funktioner 🔧
- [x] Realtidsnotiser (glöm inte att skapa en ström)
